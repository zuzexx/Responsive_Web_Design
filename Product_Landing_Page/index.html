What is this all about?

Pull Ower is a monthly subscription box for every knitter, obsessed with sweaters. Each box contains unique design, that is easy enough for first time sweater makers and fun for sweater veterans.

You can expect quality yarn and easy to follow, step-by-step instructions in each month's box. All patterns run from 4yo to adult 6XL. Before buying the subscription box, you will have to specify your desired sweater size, so we can ajust the yarn accordingly. You can change the size anytime before shipping.
